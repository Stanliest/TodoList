## Todo list (Vue.js, Express, Node.js, MongoDB)

Everything is set up on the localhost.

### Project setup
`npm install`

### Frontend
`npm run serve` localhost:8080

### Backend
`cd backend/`\
`nodemon server.js`

### DB
`docker container run -p 127.0.0.1:27017:27017 --name mongodb  -t mongo`
